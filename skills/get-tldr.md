# Get TLDR - 一句话摘要

> 将长文本浓缩为简短摘要

## 📋 技能信息
- **名称**: Get TLDR (Too Long; Didn't Read)
- **类型**: 文本处理
- **触发词**: 摘要、总结、tl;dr、tldr、太长不看、一句话概括

## 🚀 功能特点
- 提取核心信息
- 生成简洁摘要
- 保持原意准确
- 支持多长度选择

## 💡 使用示例
```
用户：tl;dr 这篇文章
助手：📝 TL;DR: 文章主要讲述了... (20 字)

用户：用一句话说完
助手：💡 核心观点：...
```

## 📊 摘要模式
- **一句话**: 10-20 字
- **短摘要**: 50-100 字
- **中标题**: 200-300 字
- **详细版**: 500+ 字
