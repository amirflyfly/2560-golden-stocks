# TDD Guide - 测试驱动开发指南

> 指导进行测试驱动开发（TDD）

## 📋 技能信息
- **名称**: TDD Guide
- **类型**: 开发方法
- **触发词**: TDD、测试驱动、写测试、单元测试、测试用例

## 🚀 TDD 流程
1. **Red** - 编写失败的测试
2. **Green** - 实现功能使测试通过
3. **Refactor** - 重构代码优化结构

## 💡 使用示例
```
用户：用 TDD 方式实现一个加法函数
助手：📝 步骤 1: 编写测试
     1. 定义测试用例
     2. 运行测试（应失败）
     
     ✅ 步骤 2: 实现功能
     实现加法函数
     
     🔄 步骤 3: 重构优化
     检查代码质量
```

## 📦 测试框架
