# Debug Pro - 代码调试助手

> 专业的代码调试和错误排查工具

## 📋 技能信息
- **名称**: Debug Pro
- **类型**: 开发工具
- **触发词**: 调试、debug、排查错误、代码问题、为什么报错

## 🚀 功能特点
- 分析错误日志
- 定位代码问题
- 提供修复建议
- 逐步调试指导

## 💡 使用示例
```
用户：这段代码为什么报错？
助手：🔍 正在分析代码...
     问题：变量未定义
     位置：第 15 行
     建议：添加变量声明
```

## 🛠️ 调试步骤
1. 重现错误
2. 定位问题
3. 分析原因
4. 提供解决方案
5. 验证修复
