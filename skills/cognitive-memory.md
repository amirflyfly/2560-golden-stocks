# Cognitive Memory - 记忆机制

> 为 AI 助手提供长期记忆和上下文管理能力

## 📋 技能信息
- **名称**: Cognitive Memory
- **类型**: 记忆系统
- **触发词**: 记忆、记住、回忆、上下文、长期记忆

## 🚀 功能特点
- 长期记忆存储
- 上下文关联
- 记忆检索
- 记忆更新和遗忘

## 💡 使用示例
```
用户：记住我喜欢吃辣的
助手：✅ 已记录：用户偏好 - 喜欢吃辣

用户：我之前说过喜欢吃什么？
助手：🧠 根据记忆，你喜欢吃辣的食物
```

## 📦 记忆类型
- **短期记忆**: 当前会话
- **长期记忆**: 持久化存储
- **工作记忆**: 任务相关
- **情景记忆**: 特定事件
