# 定时任务技能

> 创建、管理和执行定时任务，支持 Cron 表达式和自然语言

## 📋 技能信息

- **名称**: Cron Job Manager
- **类型**: 自动化工具
- **触发词**: 定时、cron、计划任务、每小时、每天、提醒

## 🚀 使用方法

### 在对话中使用
当用户需要定时执行某些任务时：
- "每小时检查一次邮件"
- "每天早上 9 点提醒我开会"
- "每天午夜备份数据"
- "每周一发送周报"

### 示例
```
用户：每天早上 9 点提醒我查看邮件
助手：✅ 已创建定时任务：每天 09:00 执行"查看邮件"
     Cron: 0 9 * * *
     任务 ID: cron_001

用户：我的定时任务
助手：
⏰ 定时任务列表:
1. 每天 09:00 - 查看邮件 (cron_001) ✅运行中
```

## 📦 安装方式

### 方法 1：使用 node-cron（推荐）
```bash
npm install node-cron
```

```javascript
const cron = require('node-cron');

// 创建定时任务
const task = cron.schedule('0 9 * * *', () => {
  console.log('每天早上 9 点执行');
  // 执行任务逻辑
});

task.start();
```

### 方法 2：使用 node-schedule
```bash
npm install node-schedule
```

```javascript
const schedule = require('node-schedule');

// 创建定时任务
const job = schedule.scheduleJob('0 9 * * *', () => {
  console.log('执行任务');
});

// 或自然语言
// const job = schedule.scheduleJob('every day at 9am', () => { ... });
```

## 📊 Cron 表达式示例

| 表达式 | 说明 |
|--------|------|
| `* * * * *` | 每分钟 |
| `0 * * * *` | 每小时 |
| `0 0 * * *` | 每天午夜 |
| `0 9 * * *` | 每天早上 9 点 |
| `0 0 * * 0` | 每周日凌晨 |
| `0 0 1 * *` | 每月 1 号 |
| `0 0 1 1 *` | 每年 1 月 1 号 |
| `*/5 * * * *` | 每 5 分钟 |
| `0 9-17 * * *` | 工作日 9 点到 17 点每小时 |

## ⚠️ 注意事项

1. **时区处理**: 确保使用正确的时区
2. **任务持久化**: 重启后任务仍存在
3. **错误处理**: 任务失败要有重试机制
4. **性能考虑**: 避免过多高频任务
5. **日志记录**: 记录执行历史便于排查
